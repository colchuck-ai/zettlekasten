# ZK skill assets

## zk-template/

Minimal layout for a **Zettelkasten**: inbox, method doc, note template, README + AGENTS for agents.

**When to use:** User wants a slip-box folder only, or “initialize zettelkasten / agent memory notes.”

**What to copy:** Everything inside `zk-template/` into the target directory (not the wrapper folder itself), so the destination gets:

- **README.md** — Human overview
- **AGENTS.md** — Agent procedures and conventions
- **zettelkasten.md** — Full method (principles, linking, structure)
- **template-zettel.md** — Body template for new notes
- **inbox/** — Quick captures (keep empty or with `.gitkeep`)

See [SKILL.md](../SKILL.md) section “Bootstrap from template.”
